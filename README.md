# Sales Predictor using Streamlit & Linear Regression

This project predicts **future sales values** using a **Linear Regression model** and lets users interactively input feature values via a **Streamlit app** to get predictions in real time.

---

## 📊 Project Overview
This app uses a **Linear Regression model** to learn the relationship between advertising data and sales. Users can input new data through a friendly interface and see predicted sales instantly. This type of model is common in business analytics and forecasting. :contentReference[oaicite:0]{index=0}

---

## 📁 Dataset
- File: `Advertising.csv`
- Features usually include variables like:
  - **TV advertising spend**
  - **Radio advertising spend**
  - **Newspaper advertising spend**
- Target variable: **Sales**  
The Linear Regression model learns how these features influence sales.
